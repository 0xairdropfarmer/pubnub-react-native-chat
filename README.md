> Build chat app with React native giftedchat and pubnub

## Install

```sh
git clone https://github.com/krissnawat/pubnub-react-native-chat
yarn or npm install
```

## Usage
add your key 
```jsx
this.pubnub = new PubNubReact({
      publishKey: "your key",
      subscribeKey: "your key",
 });
```
```sh
react native run-android or run-ios
```

## Run tests

```sh
npm run test
```
